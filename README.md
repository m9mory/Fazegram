# Fazegram

Minimalistic, clean, and customized Telegram client for iOS.

## Features
* **Pure Performance**: Based on the native Telegram-iOS source code.
* **Minimalistic UI**: Simplified interface focusing on flat design, centered geometry, and zero clutter.
* **Custom Customization**: Built for custom home screen environments and rootless setups.

## Build Setup
The project is built automatically via GitHub Actions using Bazel. 
Final output provides an unsigned `.ipa` and `DSYMs` package.
